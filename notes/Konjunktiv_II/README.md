
# Konjunktiv II (K-II)

* It makes **sentence more polite**

## When do we use K-II?
* Wir benutzen K-II, wenn wir:
* * eine Bitte machen (request)
* * ein Befehl machen (command)
* * etwas bestellen
* * etwas planen
* * uns etwas vorstellen (imagine something)

### What changes it does in the sentence?
* K-II changes the forms of helping verbs e.g:
```
haben -> hätten
sein -> wären
werden -> würden
können -> könnten
müssen -> müssten
sollen -> sollten
wollen -> wollten
```
Example sentences: 
```
1. Können Sie mir helfen? (without K-II)

Könnten Sie mir helfen? (with K-II)

2. Wirst du es für mich tun? (without K-II)

Würdest du es für mich tun? (with K-II)

3. Sie sollen zum Arzt gehen. (without K-II)

Sie sollten zum Arzt gehen. (without K-II)


wäre = will be (request)

Es wäre sehr nett von Ihnen, wenn... (It will be very help of you, if ...)

Wie wäre es am Freitag (How about on friday? / How you see on Friday?/ How you look at Friday?)

hätten = will have (request)

Ich hätte gerne einen Kaffee (I will have a coffee)

```

### Konjunktiv in Future (Imagination)

**Rule:** `werden -> würden`

**Example:**
```
1. Wir würden uns treffen

2. Ich würde eine Vila kaufen, wenn ich sehr reich sein würden.
```
### Kunjunktiv in Past (In form of Regret)

#### Partizip Perfekt

* **Helping Verbs in PP**: wären / hätten 

**Example:**
```
1. I would have stayed more, If I had met you earlier

Ich bin länger geblieben,  wenn ich dich früher getroffen hätte (without KII)

Ich wäre länger geblieben, wenn ich dich früher getroffen hätte (with KII)

2. Ich bin nach Frankreich geflogen. Ich habe die Flugkarte pünktlich gebucht

Ich wäre nach Frankreich geflogen, wenn ich die Flugkarte pünktlich gebucht hätte.
```
